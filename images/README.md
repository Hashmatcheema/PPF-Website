# PPF images

Replace placeholder images with your own assets:

- **Hero:** Use a high-resolution image (e.g. 1920×1080) for the hero section. Update the `src` in `index.html` (`.hero-bg .hero-img`) to point to `images/hero.jpg` (or your filename).
- **Team:** Add team photos as `team-1.jpg`, `team-2.jpg`, `team-3.jpg` (or more) and update the `<img>` tags in the Team section.
- **Map:** Optional map image for “Where We Exist”; update the `.map-placeholder .map-img` source in `index.html`.

Recommended: JPEG or WebP, optimized for web (e.g. 80–85% quality).
